## **DARWIN - Frontal Único de Ventas (FUV) de Naturgy**

**DARWIN** es el **Frontal Único de Ventas** (FUV) de Naturgy, un sistema integral de contratación de servicios energéticos que centraliza y gestiona todo el proceso de ventas de la compañía.

### **Características Principales:**

**1. Sistema de Contratación Multicanal**
- Gestiona ventas a través de diferentes canales: telefónico, presencial y web
- Soporta tanto contratación asistida (con agente) como no asistida (autoservicio)

**2. Integraciones con Sistemas Corporativos**
- **Siebel y Salesforce**: Sistemas transaccionales de Naturgy
- **AVIVA**: Sistema de firma digital
- **E-Voluciona**: Gestión de procesos
- **SIRCA**: Otros sistemas corporativos
- **MuleSoft**: Plataforma de integración mediante Anypoint

**3. Funcionalidades Clave:**
- Búsqueda y normalización de direcciones
- Validación de puntos de suministro (CUPS)
- Gestión de scoring y validaciones
- Manejo del bono social
- Configuración de tarifas (Tarifa Plana, Batería Virtual)
- Gestión de pedidos y contratos
- Control de flujos de envío a sistemas transaccionales (semáforo)

**4. Gestión de Agencias y Canales**
- Alta y gestión de agencias de ventas
- Configuración de canales de venta
- Integración mediante APIs REST y SFTP
- Gestión de locuciones telefónicas

**5. Infraestructura Técnica**
